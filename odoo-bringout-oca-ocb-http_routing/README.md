# Web Routing


Proposes advanced routing options not available in web or base to keep
base modules simple.


## Installation

```bash
pip install odoo-bringout-oca-ocb-http_routing
```

## Dependencies

- web

## Source

- Repository: https://github.com/OCA/OCB
- Branch: 16.0
