# mail_bot_hr

This module adds the OdooBot state and notifications in the user form modified by hr.

## Installation

```bash
pip install odoo-bringout-oca-ocb-mail_bot_hr
```

## Dependencies

This addon depends on:
- mail_bot
- hr
