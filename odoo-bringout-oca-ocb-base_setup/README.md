# Initial Setup Tools


This module helps to configure the system at the installation of a new database.
================================================================================

Shows you a list of applications features to install from.

    

## Installation

```bash
pip install odoo-bringout-oca-ocb-base_setup
```

## Dependencies

- base
- web
