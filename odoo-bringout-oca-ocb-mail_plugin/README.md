# Mail Plugin

Integrate Odoo with your mailbox, get information about contacts directly inside your mailbox, log content of emails as internal notes

## Installation

```bash
pip install odoo-bringout-oca-ocb-mail_plugin
```

## Dependencies

This addon depends on:
- web
- contacts
- iap
