# WMS Accounting


WMS Accounting module
======================
This module adds the price difference account. Used in standard perpetual valuation.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-purchase_price_diff
```

## Dependencies

- purchase_stock
